How to find X/Y Coordinates for ADB shell input tap Command

First we need to connect the device to the adb shell and execute adb devices to finde the device.

In the next step we can capture the current screen on the phone with adb shell screencap -p /sdcard/screencap.png.

This will save the screenshot on the internal storage so we need to download it onto the pc with adb pull /sdcard/screencap.png.

Now we cann analyse it with paint and find the x and y coordinates to execute the tap command.

The tap command adb shell input tap x y can be executed now.
