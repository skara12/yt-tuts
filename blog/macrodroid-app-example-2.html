MacroDroid App: Example 2

With this Example we can unlock the Screen of any Android Phone.

As Trigger we will use Battery/Power when connected.

For the Action to be performed we can use a Shell script:

input swipe 500 1700 500 100
sleep 3
input text 1479
sleep 3
input keyevent 66

This will Type in the password for the lockscreen and unlock it when power is connected.
