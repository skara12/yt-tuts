Send SMS from Adb Shell

Use Android services to send sms from adb shell

1. Enable USB Debugging in the Settings

2. Download Platform Tools https://developer.android.com/studio/releases/platform-tools

3. Connect Phone to PC and start Command Line in the Platform Tool Folder

4. Type adb devices

Parameters for service call isms:

7 (is the position of the Function in the file from github: https://github.com/aosp-mirror/platform_frameworks_base)

i32 0 (is the sim card slot, it can be 0 , 1 ,2 )

str16 "com.android.mms.service" (is the package name of the service)

str16 "+1234567890" (is the receiver phone number)

str16 "null" (scAdress can be 0 for Android 5+ )

str16 "test" (is the message)

next parameters can be "null"
