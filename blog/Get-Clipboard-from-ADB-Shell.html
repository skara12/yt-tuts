Get Clipboard from ADB Shell

To get the clipboard we need to call the command adb shell service call clipboard 2 s16 com.android.shell

This will give a hidden output so we need to save the output in a file with adb shell service call clipboard 2 s16 com.android.shell > clipboard.txt

The python code will help to get the direct output:

#opens the outputfile file
raw = open("clipboard.txt", "r")
str = ""
i = 0
#read all lines
out =raw.readlines()

#loops through all lines
for line in out :
   s = line.split("'")
   #if the array is more than 1 string we can store the second string
   if len(s) > 1:
    str+=s[1]
    #make spaces
    if i < 4:
     str+='\t'
    i+=1
# print the clipboard
print(''.join(str.split(".")))
