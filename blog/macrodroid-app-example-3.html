MacroDroid App: Example 3

With this Example we can swipe up when a floating button is pressed.

As Trigger we will use User Input Floating Button.

For the Action to be performed we can use a Shell script:

input swipe 300 1000 300 200

This code makes a swipe from 300,1000 to 300,200.
